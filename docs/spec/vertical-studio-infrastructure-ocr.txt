Vertical Studio - Kompletni infrastruktura

1. Pfehled systému

Vertical Studio je automatizovany systém pro generovani marketingovych webti pro boutique developery. Systém
transformuje strukturovana JSON data o firmé do piné funkéniho, nasaditelného webu.

1.1 Hlavni cile

Rychlost: Generovani webu do 5 sekund
Skalovatelnost: 100+ webii masiéné

Kvalita: Premium design odpovidajici cenové kategorii
Individualita: KaZdy web je unikatni (ne Sablonova kopie)
